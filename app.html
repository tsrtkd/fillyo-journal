👥 강사 초대 (Pro)
강사 이메일로 초대하기
초대된 강사는 같은 학원 데이터를 실시간 공유합니다.
강사가 fillyo에 가입 후 초대 코드를 입력하면 연결됩니다.
학원 코드 (강사에게 공유)
📖 사용 가이드
1
이달 총원 입력
매달 1일, 화면 하단 이달 인원 변동 등록에서 기준 총원 입력.
2
매일 업무일지 작성 후 저장
부별 출석 입력 → 결석 자동계산. 저장 시 팀원에게 알림 전송.
3
캘린더로 날짜 이동
헤더 📅 버튼 → 달력에서 날짜 선택. 데이터 있는 날은 파란색 표시.
4
홈 화면에 앱처럼 추가
아이폰: Safari → 공유(□↑) → 홈 화면에 추가
안드로이드: Chrome → ⋮ → 홈 화면에 추가
